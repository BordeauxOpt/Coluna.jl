# Coluna

## coding style

- use `_` split words in a given name when there is at least one uncommon shortcut. Common shortcuts are: 
ref(refrence), 
var(variable), 
constr(constraint), 
lb(lower bound), 
ub(upper bound), 
form(formulation), 
prob(problem), 
cur(current), 
rhs(right hand side), 
lhs(left hand side), 
val(value), 
col(column), 
coef(coefficient), 
ip(integer program), 
lp(linear program), 
inc(incumbent), 
sol(solution), 
eval(evaluation), 
alg(algorithm), 
info(information)
